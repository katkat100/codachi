# Codachi Improvement Ideas

## Gameplay Features

### New Interactions
- [ ] **Play** - A mini-game or animation that boosts happiness
- [ ] **Sleep** - Put pet to bed to restore health faster (but can't interact while sleeping)
- [ ] **Treats** - Special items bought with points for bonus effects

### More Progression
- [ ] **Skins/Themes** - Unlock different color schemes or pet variants
- [ ] **Hats/Accessories** - Cosmetic unlocks from achievements
- [ ] **Prestige system** - Reset at max level for permanent bonuses

## Code Integration

### Smarter Reactions
- [ ] React to specific error types differently (syntax vs type errors)
- [ ] Celebrate when tests pass
- [ ] Get excited when you create new files
- [ ] Track lines of code written per session

### Language Support
- [x] **Auto-detect project type** (Rust, JS, Python, etc.)
- [x] **Built-in lint configs** for common languages
- [ ] React to `TODO` and `FIXME` comments

## Social Features

### Multiplayer
- [ ] See friends' pets in a shared view
- [ ] Pet playdates when both are online
- [ ] Compare stats/achievements on a leaderboard

### Sharing
- [ ] Export pet as ASCII art to share
- [ ] Generate shareable pet cards/stats

## Quality of Life

### UI Improvements
- [ ] Sound effects (optional)
- [ ] More animation frames for smoother movement
- [ ] Weather/time-of-day backgrounds
- [ ] Pet thought bubbles with more personality

### Notifications
- [ ] Desktop notifications when pet needs attention
- [ ] Daily summary of pet stats

### Persistence
- [ ] Cloud sync for pet state across machines
- [ ] Pet history/timeline of major events
